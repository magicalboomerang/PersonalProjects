How to hand build a GUI:

1. 4 Variables for the display
	1. Max child count
	2. Given child count
	3,4. Display resolution

2. 6 Variables for system layer
	1. Max child count
	2. Given child count
	3,4. Location
	5,6. Layer resolution
	
Everything else will loop through these steps as needed.

3. Screens - 2 vars
	1. Max child count
	2. Given child count

4. Layers - 6 vars
	1. Max child count
	2. Given child count
	3,4. Location
	5,6. Dimensions

5. Images - 4 Vars + data
	1,2. Location
	3,4. Dimensions
	5. Data
